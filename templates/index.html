How It Works

Our Model

This application uses Facebook's Prophet, a powerful forecasting tool that implements time-series forecasting procedures. The model accounts for:

  • Yearly, weekly, and daily seasonality
  • Holiday effects
  • Trend changes and outliers

The result is a robust prediction system that adapts to your website's unique traffic patterns.

Getting Started

  2. Try different prediction periods on the Predict page
  3. Upload your own CSV data to get customized predictions

Your data should include a date column in YYYY-MM-DD format and a pageviews column with numerical values.
